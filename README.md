# Claude Memory Template

A personal `CLAUDE.md` template that gives Claude Code persistent memory about who you are, how you write, and what you're working on — without re-explaining yourself every session.

## What this is

Claude Code reads a `CLAUDE.md` file from your home folder at the start of every session. This template gives you a structured starting point to fill in:

- Who you are and your background
- How you like to communicate
- Your writing voice (including words and vibes to avoid)
- Your active projects and context
- How you work with code

## How to use it

1. Download or copy `CLAUDE.md` from this repo
2. Fill in your own details (takes ~15 minutes)
3. Move it to your home folder:
   ```bash
   mv CLAUDE.md ~/CLAUDE.md
   ```
4. Open Claude Code — it will read the file automatically from now on

## Pro tip: port your Claude.ai memory

If you've been using Claude.ai (the website), you can export your conversation memory and have Claude Code read it to pre-fill your profile:

1. Go to **claude.ai → Settings → Privacy → Export data**
2. Download the ZIP file
3. Open Claude Code and say: *"Read this export and save the key facts to memory"*
